# Monsterbation Configuration Linter

An online tool to check your configuration for [Monsterbation](https://forums.e-hentai.org/index.php?showtopic=211039).

## Contribute

### Install

```bash
$ npm i
```

## Development

```bash
$ npm run build-eslint
$ npm run serve
```

## Build

```bash
$ npm run build-eslint
$ npm run build
```

## Clean Up

```bash
$ npm run clean
```
